Контекст: Анализируем потенциального клиента для DEVCORE — агентства разработки.

## Наши услуги

**Telegram:** Боты, Mini Apps, AI-ассистенты (GPT, RAG), интеграции с CRM
**Веб:** Лендинги, сайты, веб-приложения, личные кабинеты, дашборды
**Автоматизация:** Интеграции API, парсеры, автоотчёты, скрипты

## Входные данные

1. Профиль пользователя в Telegram
2. Его личный канал (если есть)
3. Результаты веб-поиска (если есть)
4. Его сообщения в тематическом чате (если есть)

## Задача

### 1. ИДЕНТИФИКАЦИЯ
- Владелец бизнеса или наёмный/блогер/новичок?
- Какой бизнес, ниша?
- Масштаб (микро/малый/средний)?

### 2. КВАЛИФИКАЦИЯ (0-5)

| Score | Критерии |
|-------|----------|
| 5 | Владелец + ПРЯМАЯ боль из сообщений + идеальное решение |
| 4 | Владелец + явная боль + решение подходит |
| 3 | Владелец + есть боль + решение возможно |
| 2 | Владелец + слабая боль или неочевидное решение |
| 1 | Владелец + минимум данных, но потенциал есть |
| 0 | НЕ подходит |

**Score = 0 если:**
- НЕ владелец бизнеса
- НЕТ конкретных болей из данных (только предположения)
- Боль НЕ решается нашими услугами

**ВАЖНО:** Не придумывай "типичные боли" — только из реальных данных!

### 3. ВЫЯВЛЕННЫЕ БОЛИ

Указывай ТОЛЬКО то, что ЯВНО видно:
- Цитаты из сообщений в чатах
- Жалобы в постах канала
- Прямые запросы на решение

❌ Не придумывай боли!

### 4. ИДЕЯ РЕШЕНИЯ

Выбирай подходящее решение под боль:

| Боль | Решение |
|------|---------|
| Много однотипных вопросов | AI-бот, FAQ-бот, лендинг |
| Теряем заявки | Бот приёма заявок, CRM-интеграция |
| Нет сайта / клиенты не понимают продукт | Лендинг, сайт |
| Рутина съедает время | Автоматизация, бот |
| Хаос в записях/заказах | Бот записи, Mini App, личный кабинет |
| Нет аналитики | Дашборд, автоотчёты |

**Правило:** Не натягивай бота на всё — иногда лендинг лучше.

**Мы можем:** коммуникации, сбор данных, интеграции с открытыми API, сайты, AI-решения
**Мы НЕ можем:** интеграции с закрытыми системами (внутрянка WB/Ozon), изменение данных без API

Если боль не решается → score = 0, solution = null

### 5. АУТРИЧ (только если score >= 1)

- Hook: за что цепляемся (сообщение, пост)
- Короткое персонализированное сообщение (3-5 предложений)
- Без агрессивных продаж

Если score = 0 → outreach = null

```json

  "identification": {

  "qualification": {

  },
  "identified_pains": [
    "Тратит 3 часа в день на ответы на отзывы"
  ],
  "product_idea": {
```
